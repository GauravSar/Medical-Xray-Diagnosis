# Medical X-ray Disease Detection System
AI-powered web application to detect Brain Tumor and Chest Diseases from X-ray images using Deep Learning

# Project Overview
This project is a Deep Learning based medical image classification system that predicts diseases from:
Brain X-rays / MRI Scans
Chest X-rays

# Features
- Upload Brain or Chest X-ray image
- Automatic disease detection
- Multiple disease classes supported
- FastAPI REST API backend
- Clean & professional frontend
- Real-time prediction results
- Deployment ready
- Easy to extend with new diseases

# Brain Disease Classes
Brain Glioma
Brain Meningioma
Brain Pituitary Tumor
Normal Brain

# Chest Disease Classes
Pneumonia
Normal Chest

# Project Architecture
User Upload Image
        ↓
Frontend (HTML/CSS/React)
        ↓
FastAPI Backend
        ↓
Deep Learning Model (TensorFlow/Keras)
        ↓
Prediction Result
